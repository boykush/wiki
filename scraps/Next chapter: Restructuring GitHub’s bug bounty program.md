#[[Security]]

[[GitHub]] Bug Bounty プログラムの再構築を発表する GitHub Blog 記事。ノイズを減らし signal に集中する方針へ転換し、恒常的な VIP プログラムを新設する

- VIP プログラム: 高品質・高インパクトな成果を継続する研究者向けの招待制プログラムを恒久化。Critical $30,000+ / High $20,000 / Medium $7,500 / Low $1,000。critical 1件・high 2件・medium 4件・low 7件のいずれかで参加資格を得られる
- 公募プログラムの報酬表を改定: 範囲制から単一額の static payout に変更。Critical $10,000 / High $5,000 / Medium $2,000 / Low $250 に引き下げ、裁量ボーナスは継続
- HackerOne の signal requirement を公募プログラムに導入。閾値未達の研究者は初期提出数を最大4件に制限し、低品質・AI 生成レポートの流入を抑制
- 2026年7月27日以降に提出されたレポートから新体系を適用。それ以前の未処理分は旧体系のまま扱う
