#[[Continuous Integration]]

[[GitHub]] の最上位プラン GitHub Enterprise を、GitHub がホスト（github.com 上の SaaS）する形態で提供するもの。略称 GHEC。もう一方の提供形態は self-hosted の GitHub Enterprise Server (GHES)。

- 複数 organization を束ねる「enterprise アカウント」を持ち、ポリシー・課金・ruleset を enterprise レベルで一元管理できる
