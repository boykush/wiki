#[[Cloud Infrastructure]]

トラフィックを複数のコンピュートリソースにルーティングし分散させ、システムの可用性向上、フォールトトレランスを行うサービスの集まり

リソースごとに以下のような種別がある

- Application Load Balancer
- Network Load Balancer
- Gateway Load Balancer
