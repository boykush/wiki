#[[Security]]

[[GitHub]] が管理するオープンソース脆弱性の公開データベース。CVE ID の採番申請（CNA）、GHSA のキュレーション、Dependabot アラートへのデータ提供を統括する。

- 2026年5月の月間レビュー済みアドバイザリは 1,560 件（通常の5倍超）
- Private Vulnerability Reports は 2026年1月の〜550件/週から5月に 3,000件/週超へ急増
- CVE 申請数は前年比約10倍（2026年5月単月で約 4,000 件超）
- キュレーター業務：パッケージ名の名寄せ、バージョン範囲の再構築、複数エコシステムにまたがる advisory の調整
- CVSS 3.1 / 4.0 と CWE による severity 分類
