Static Application Security Testing

#Security #Testing

ソースコード、バイトコード、バイナリを解析してセキュリティ脆弱性を検出するテスト手法

ホワイトボックステストとも呼ばれ、アプリケーションを実行せずにコードを分析する

特徴

  • DevSecOpsのシフトレフトを実現
  • CIパイプラインに統合可能
  • SQLインジェクション、XSS、バッファオーバーフロー等を検出

DAST（動的テスト）と組み合わせて使用されることが多い
