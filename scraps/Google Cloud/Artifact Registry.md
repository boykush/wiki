#[[Cloud Native]] #[[Security]] #[[Continuous Integration]]

[[Google Cloud]]が提供する[[コンテナ]]イメージや言語パッケージを保管・管理するレジストリサービス

Dockerコンテナイメージ ([[Open Container Initiative|OCI]])、Helmチャート、Go/Java/Node.js/Python/Rubyパッケージ、Debian (Apt) / RPM OSパッケージなどに対応する

Container Registryの機能を拡張した後継サービスとして、[[Google Cloud]]で推奨されるコンテナレジストリと位置づけられている
