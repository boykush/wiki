#[[Security]]

"Unified Security Platform" を標榜する商用 SaaS（カテゴリは ASPM / Application Security Posture Management）。10 種類超のセキュリティスキャンを単一 UI に集約する

スキャン対象: [[SAST]] / SCA / [[DAST]] / シークレット / [[Infrastructure as Code|IaC]] / [[コンテナ]] / CSPM / マルウェア / Code Quality / Outdated Software

Terraform / CloudFormation / [[Kubernetes]] マニフェストの設定ミス検出にも対応

## 特徴

- AutoFix / Bulk Fix で修正 PR を自動生成
- AI Pentesting Agents による DAST 自動化（200+ エージェント）
- Aikido Intel - 依存パッケージのマルウェア検出
- TL;DR Summary で脆弱性の要約と対応指針を提示
