[[Open Container Initiative|OCI]]準拠のコンテナイメージレジストリサービス

Red Hatが提供する3つの形態:

- Quay.io - フルマネージドホスティング版（パブリックリポジトリは無料）
- Red Hat Quay - オンプレミス/プライベートクラウド向けエンタープライズ版
- Project Quay - Apache 2.0ライセンスのオープンソース版

主な機能:

- 脆弱性スキャン - Clairによるイメージの自動スキャン
- アクセス制御 - きめ細かい権限管理
- 自動ライフサイクル管理 - イメージの構築・配信・デプロイの自動化
